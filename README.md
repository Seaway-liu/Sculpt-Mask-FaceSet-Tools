# Sculpt-Mask-FaceSet-Tools
This is a Blender sculpting support tools!  
这是一个Blender雕刻辅助工具! 

https://github.com/Seaway-liu/Sculpt-Mask-FaceSet-Tools/blob/main/Sculpt_Mask_FaceSet_tools_EN.jpg

An easy-to-use, user-friendly Sculpt Pie Menu support add-on for everyone from beginners to professionals.
The tool design emphasizes clarity and efficiency, combining all the Mask and Face Sets commands scattered across the sculpting menu into a Pie Menu for convenient and quick access.

Features:

• The left Menu contains all commands for Mask.

• The right Menu contains all commands for FaceSet.

• The top Menu includes transformation and display management.

• The bottom Menu includes Mesh Filter, Mask Extract, and Slice.


This Addon supports switching between Chinese and English languages, and defaults to the Blender system language.


• Minimum Blender version: 4.5

• Default shortcut key: 1

Usage:
1. Select the Mesh object and enter sculpting mode,
2. Press the default shortcut key: 1 to bring up the menu.

Note: The author does not have the conditions to test whether the "macOS-arm64" and "Linux-x64" operating systems can be used normally.
